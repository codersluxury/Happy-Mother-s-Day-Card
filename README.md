# Happy Mother's Day Card

A simple HTML-based flip card greeting for Mother's Day.

## Overview

This folder contains a single-page card design built with HTML and CSS. It displays a front cover with a Mother's Day message and flips over on hover to reveal the inside greeting.

## Files

- `index.html` — the complete card page with inline CSS and animation.

## Features

- 3D flip card effect on hover
- Soft pink gradient styling
- Animated floating heart icons
- Inside greeting message for Mother's Day

## Usage

1. Open `index.html` in any modern web browser.
2. Hover over the card to flip it and reveal the inside message.

## Notes

- No external dependencies are required.
- The card is fully self-contained in a single HTML file.
